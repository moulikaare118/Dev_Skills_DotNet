# DevSkills Assessment

A responsive React + Vite application that simulates a DevSkiller-style coding assessment flow.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev -- --host
```

3. Open the URL shown in the terminal.

## Features

- Landing page with disabled dark mode toggle and acceptance checkbox
- Task selection with a single enabled workflow
- Assessment page with timer persistence, upload validation, and simulated build/test results
- Responsive layout using Tailwind CSS
